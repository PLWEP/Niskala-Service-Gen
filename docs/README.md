# Niskala Service Gen Documentation

Welcome to the official documentation for **Niskala Service Gen**, a powerful code generator for Dart and Flutter.

## Overview

Niskala Service Gen automates the creation of strongly-typed API services from OpenAPI/OData metadata, focusing on enterprise-grade features like:

- **Type-Safe Queries**: Expressive OData query builder.
- **Resilience**: Integrated retries and auth refresh.
- **Developer Experience**: Watch mode, dry-runs, and instant scaffolding.
